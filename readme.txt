== Description ==

You can add pdf, xlsx, ppt, image from Google Drive. And Imgur image hosting, Youtube, Vimeo, Dailymotion video you can embed from this plugin easily using only id.

== Installation ==

You can install this plugin easily. Just follow step by step. 

1. Go to "Plugin" from your WordPress Dashboard.
2. Then Click "Add New"
3. Click "Upload Plugin" then "choose file"
4. Select "shine-pdf-embeder.zip" this file from your download folder.
